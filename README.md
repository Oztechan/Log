## Install

Too install the library please add
```gradle
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
to build.gradle (project)

and
```gradle
	dependencies {
	        implementation 'com.github.SUB-MOB:logmob:VERSION_NUMBER'
	}
```
to build.gradle (app)
